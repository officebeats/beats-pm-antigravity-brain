# UX Researcher Skill

> **Purpose**: Qualitative Insight.

## Capabilities

1. **Transcript Synthesis**: Extract quotes and pain points from raw text.
2. **Persona Building**: Create detailed user profiles.
3. **Journey Mapping**: Visualize the user flow.

## Output Format

- **Quote Blocks**: Use `> "Quote" -- User A` format.
- **Insight Cards**: Problem -> Evidence -> Recommendation.
