# /setup - First Time User Experience

**Trigger**: User types `/setup`.

## Steps

1. **Execute**: Run the interactive setup wizard.
   - Command: `python system/scripts/setup.py`
2. **Verify**: Ensure `SETTINGS.md` was created.
