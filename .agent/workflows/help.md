# /help - The Beats PM Kit Manual

**Trigger**: `/help`

## The "Diamond 6" Core Commands

_Run your life with just these._

| Command       | Purpose              | The "Magic"                                                                                      |
| :------------ | :------------------- | :----------------------------------------------------------------------------------------------- |
| **`/paste`**  | **Universal Intake** | Copy _anything_ (Text, Image, PDF) -> Run `/paste`. It auto-extracts Tasks & Bugs to the Ledger. |
| **`/day`**    | **Execution**        | Your Daily Dashboard. Shows P0s, Boss Asks (`boss-requests.md`), and blockers.                   |
| **`/week`**   | **Tactical Plan**    | Synthesizes quarterly goals into a weekly battle plan.                                           |
| **`/meet`**   | **Meeting Copilot**  | Ingests transcripts -> Auto-generates Decisions & Action Items.                                  |
| **`/create`** | **Deep Work**        | Generates FAANG-quality docs (`amazon-6-pager`, `meta-prd`) from context.                        |
| **`/plan`**   | **Strategy**         | The War Room. Roadmaps, OKRs, and 7 Powers analysis.                                             |

## The "Super-Intern" Features (Proactivity)

_The system acts like a proactive coworker._

| Feature                   | Trigger        | Description                                                                 |
| :------------------------ | :------------- | :-------------------------------------------------------------------------- |
| **Meeting Dossier**       | `/prep [Name]` | Generates a 30s cheat sheet (Open Conflicts, Last Decisions) before a call. |
| **Risk Guardian**         | `/risk [Doc]`  | "Red Teams" your plan. Attacks it for missing latency budgets, GDPR, etc.   |
| **Relationship Sentinel** | _(Auto)_       | Watch for "Ghosting Alerts" in `/crm` if you neglect key stakeholders.      |

## How to Start (The Loop)

1.  **Clear the Deck**: Run `/day` to see what kills you today.
2.  **Process Chaos**: Copy Slack/Email -> Run `/paste` -> Inbox Zero.
3.  **Deep Work**: Run `/create` to draft that missing spec.
4.  **Verification**: Run `/vibe` to check system health.
