# ⚔️ Task & Execution Playbook

This workflow guides the **Staff PM** to manage daily execution.

## Steps

1.  **State Check**:
    - Read `5. Trackers/TASK_LEDGER.md` (or `todo.md`).
    - Read `5. Trackers/BOSS_ASKS.md`.
    - Read `5. Trackers/BUG_TRACKER.md`.

2.  **Triage**:
    - **Input**: User Brain Dump ("I need to fix X, email Y, and plan Z").
    - **Action**: Parse input and route to the correct file.
    - **Priority**: Assign P0 (Today), P1 (This Week), P2 (Backlog).

3.  **Boss Ask Protocol**:
    - If the user mentions a "Boss" or "Leadership" request, IMMEDIATELY flag as `[BOSS]` in `BOSS_ASKS.md`.

4.  **Output**:
    - Display the updated "Today's List" (Table View).
