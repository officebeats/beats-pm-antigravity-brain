# Commands Reference

All commands can be triggered as `/command` in your AI tool.

| Command | Action | Agent |
|---------|--------|-------|
| `/day` | Daily briefing and synthesis | CPO |
| `/boss` | Prepare for 1:1 with your boss | CPO |
| `/task` | Manage tasks and priorities | Staff PM |
| `/track` | Battlefield View (Tasks + Bugs) | Staff PM |
| `/meet` | Synthesize transcript to action items | Staff PM |
| `/create` | Document Factory (PRD, Spec, One-Pager) | Staff PM |
| `/plan` | Strategic War Room (Roadmaps, OKRs) | Strategist |
| `/sprint` | Sprint Plan Generation | Program Manager |
| `/retro` | Action-driven Retrospective | Program Manager |
| `/discover` | Product Discovery (OST) | Staff PM |
| `/prioritize` | Backlog Scoring (RICE, etc) | Staff PM |
| `/paste` | Capture clipboard content | Staff PM |
| `/review` | Code/Spec/Design Quality Control | Tech Lead |
| `/vacuum` | System Optimization & File Archive | CPO |
| `/fan-out` | Complex Parallel Agent Dispatch | CPO |
| `/regression` | Run full CI tests on the Kit | QA Engineer |
| `/help` | User Manual & System Docs | System |
