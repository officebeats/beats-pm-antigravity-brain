## What Is This?

A **folder structure + markdown files + AI agent prompts** that work together to:

- ‚úÖ Capture any input (emails, Slack, notes, screenshots) and organize automatically
- ‚úÖ Track boss requests and critical bugs so nothing slips
- ‚úÖ Manage stakeholder updates proactively
- ‚úÖ Collaborate with UX and Engineering teams
- ‚úÖ Synthesize patterns into strategic opportunities
- ‚úÖ Give you daily briefs so you always know what matters

**Works with**: Google Antigravity, Claude Code, Gemini CLI, Cursor, or any AI tool.
