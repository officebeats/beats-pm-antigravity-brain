# Beats Product Management Antigravity Brain

> AI-powered PM second brain. Capture chaos. Surface patterns. Never let critical items slip.

## Quick Start

1. Read `START-HERE-FTUE.md` for guided setup
2. Configure `SETTINGS.md` with your details
3. Start capturing with `#` commands
4. Generate briefs to see it in action
